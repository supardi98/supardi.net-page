# Gemini Notes

This project was generated by Gemini (Antigravity).

## Stack
- Frontend: Vue 3 + Vite
- Styling: Tailwind CSS
- Backend API: Cloudflare Pages Functions
- Database: Cloudflare D1

## Features Included
- Beautiful, mobile-first design tailored for "Link in Bio" use cases (e.g. Instagram).
- Glassmorphism effects and micro-animations.
- Serverless tracking endpoint (`/api/track`) deployed alongside the Vue app on Cloudflare Pages.
- Seamless D1 SQL database integration.

## Next Steps
To further customize the landing page:
1. Update the `links` and `profile` objects in `src/App.vue`.
2. Add a custom domain in your Cloudflare Pages project settings.
3. Replace the `database_id` in `wrangler.toml` once your D1 instance is created.
